## 📝 Descripción del Proyecto
Este repositorio contiene el desarrollo del Trabajo Práctico Integrador de Git Avanzado. El proyecto consiste en una aplicación web desarrollada exclusivamente en HTML puro, sin el uso de frameworks.

El foco de este repositorio no está en la complejidad del código, sino en la aplicación de procesos de trabajo colaborativo simulando un entorno real. Los objetivos técnicos implementados incluyen:
* **Estrategia de Ramas:** Uso de una rama `main` y ramas individuales (feature) para cada integrante.
* **Resolución de Conflictos:** Generación y resolución colaborativa de conflictos de merge.
* **Integración Continua (CI):** Implementación de flujos de trabajo con GitHub Actions para validar automáticamente la sintaxis del código HTML en cada push.
* **Versionado:** Gestión de versiones del proyecto mediante la creación de Git Tags.
